# ZZZ_Fairy

<p>This project is a local AI Voice Assistant (running on windows) that acts as a personal assistant. It can perform general AI Assistant tasks, such as general conversation and web information fetching. Additionally, it also does automation features such as email checking, discord message checking, and code automation. It also performs device control like checking performance (CPU, RAM, and Network Throughput), battery levels, and security. As a bonus, it also checks information on my Zenless Account. Characters, Stats, latest news, banners, etc. It should be noted that these features are only Fairy's base features, and will continue to expand later on. </p>

<p>Inspired by the AI-Assistant in Zenless Zone Zero: Fairy, as well as the interest of learning about AI and its various use cases and integrations in emerging technologies, this big and ambitious side project is meant for me to learn about various things related to AI (such as agentic workflows, automation, multi-model orchestration, ML and DL, etc.), emerging and existing technologies (such as System Architecture, Security, Web Scraping and Fetching, etc), and even low-level concepts (such as Concurrency, Multithreading, Starvation and Deadlock, etc.)</p>
